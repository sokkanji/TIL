# preventDefault

### preventDefault란?

a 태그나 submit 태그는 누르게 되면 href를 통해 이동하거나, 창이 새로고침하게 되는데, 이러한 동작을 막는 역할을 한다.

### 주로 사용되는 경우

1. a 태그를 눌렀을 때도 href 링크로 이동하지 않게 할 경우
2. form 안에 submit 역할을 하는 버튼을 눌렀어도 새로 실행하지 않게 하고 싶을 경우(submit은 작동됨)
